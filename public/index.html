Docco is a quick-and-dirty, hundred-line-long, literate-programming-style documentation generator. It produces HTML that displays your comments alongside your code. Comments are passed throughMarkdown, and code is This page is the result of running Docco against its own source file.

If you install Docco, you can run it from the command-line:

docco src/*.coffee

will generate linked HTML documentation for the named source files, saving it into a docs folder.

The source for Docco is available on GitHub, and released under the MIT license.
